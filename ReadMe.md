# BoAt UI

Implementation the Landing page (main page) of "BoAt" UI

## Description

The BoAt website has been replicated using simple HTML and Tailwind CSS classes.

## Set up
Clone the repo and in the folder run the below commands.
```bash
npm install
```

```bash
npm run ndev
```
